
(1)接受invitation之后克隆repo：
    在你想clone的目录下：
    git clone https://github.com/zschen211/cs411-project.git cs411git
        可以把最后的cs411git换成你自己想叫的文件夹名

(2)更新本地repo，一定加rebase!!!!
    git pull –-rebase

(3)本地有改动之后想更新到github上：
    git add .
    git commit -m "san值高的话可以写得清楚些"
    git push
        记得push之前在群里说下

(4)多人开发的话，想commit/push可能会提示conflict， 把conflict resolve就行了
    跟上一个push的队友对下有什么区别，再决定要不要push
